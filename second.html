Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM.For example :
  1. GetElementById() = We can select an element by their id with GetElementById().
  2. GetElementByClasses() = We can select an element by their class with GetElementByClasses().
  3. GetElementByTagName() = We can select an element by their tag name with GetElementByTagName().
  4. querySelector() = We can get a element by their css selector by querySelector().It return only one element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we use event listener on parent element . Its make easy to get its child element unless we have use event listener each of a child element.

Question-4: What is event bubbling in the DOM?

When we creat an event to a child element. Its goes down to top like, child - parent - grandparent - document and its called event bubbling.
